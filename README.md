## 核心理论：Ferry

以人类幸福为总导向(不同人有不同样的幸福标准)，双方需求产生关系，保障人类的安全、质量、效率为目标效果，以降低成本为具体衡量标准。差距驱动活动规划，不断同步实际信息与建模信息，掌握工具属性，匹配修正错误，以达成目标。

需求决定行动，行动产生关系，实践中:不断对齐认知修正错误，达到目标。
