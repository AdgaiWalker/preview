## 核心理论：Ferry

**Ferry（渡）** 是一套从实践中总结的 AI 协作方法论，核心主张：

- **人类主权优先**：AI 的目的是服务人的幸福，而非替代或压榨
- **权力分配清晰**：明确人做什么、AI 做什么
- **安全与效率兼顾**：在工程实现中保障信息安全
- **跨场景适用**：编程、内容创作、具身智能——任何 AI 协作场景

> 理论详情见 [Ferry 仓库](https://github.com/AdgaiWalker/Ferry)
